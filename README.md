# Ping Pong Game Site

A simple browser-based ping pong game built with HTML, CSS, and JavaScript.

## How to run

1. Open `index.html` in your browser.
2. Press the "Start / Restart" button.
3. Use `W` / `S` to move the left paddle.
4. The right paddle is controlled by the CPU.

## Files

- `index.html` — game layout and controls
- `styles.css` — visual styling
- `script.js` — game logic and animation
