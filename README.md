# High Ping 🏓

A simple browser-based ping pong game where you play against the computer with adjustable difficulty levels.

This is my first project on GitHub, inspired by my experience at a hackathon. I'm a beginner exploring tech and learning by building projects like this.

---

## 🎮 Features

* Play against a computer opponent
* Adjustable difficulty modes
* Simple keyboard controls
* Runs directly in the browser

---

## ▶️ How to Play

1. Open `index.html` in your browser
2. Click **Start / Restart**
3. Use:

   * `W` → Move paddle up
   * `S` → Move paddle down
4. Try to beat the computer!
